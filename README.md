# Olympic-Race-Simulator

URL: https://www.youtube.com/watch?v=WtWILVYVZ9A

Description:
The Olympic Race Simulator is an engaging and interactive web-based application designed to recreate the excitement of a sprint race between multiple countries in a visually dynamic way. Developed using core front-end technologies—HTML for structure, CSS for styling, and JavaScript for functionality—the project brings together animation, user input, and real-time updates to produce a lively and immersive racing experience. It is both a fun simulation and a practical demonstration of key web development concepts.

At the start, users are presented with a selection menu containing a variety of countries. Each country is paired with its national flag, adding a recognizable and authentic visual element to the interface. Users can select multiple countries at once, allowing for customizable races with different participants each time. Once the selection is made, the “Create Race” feature generates a track where each chosen country is assigned its own lane. Behind the scenes, each participant is also given a randomly generated speed, ensuring that no two races are ever the same and keeping the outcomes unpredictable and exciting.

Before the race begins, a countdown sequence appears on the screen, building anticipation and mimicking the structure of a real sporting event. When the countdown reaches zero, the race starts, and animated runner icons begin moving across the track toward a clearly marked and visually enhanced finish line. The motion of each runner is controlled dynamically, creating a sense of competition as they progress at different speeds.

As the race unfolds, each lane displays a live timer that continuously updates, giving users real-time insight into the performance of each participant. When a runner reaches the finish line, their movement stops, their final time is recorded, and they are placed into a leaderboard. The leaderboard updates instantly, ranking competitors based on their finishing order. To further enhance the competitive atmosphere, medals are awarded to the top three finishers—gold for first place, silver for second, and bronze for third.

The application also includes intuitive controls that allow users to easily create, start, and reset races, making it simple to run multiple simulations in succession. Visually, the project is enriched with features such as animated runner GIFs, glowing and color-changing title text, and a pulsing finish line effect, all of which contribute to a more polished and engaging user experience.

Overall, the Olympic Race Simulator effectively combines interactivity, animation, and real-time data handling to create a compelling and educational web application that showcases the power and flexibility of front-end development.
